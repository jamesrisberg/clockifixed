# Endpoint Reference

All endpoints are accessed through the `Clockify` facade. Each domain is a property containing typed methods. All response types are [unified](/sdk/type-system) — a `Project` is a `Project`, regardless of which operation returned it.

## `clockify.timeEntries` — TimeEntry

| Method | Returns | Description |
|---|---|---|
| `create(body)` | `TimeEntry` | Create a time entry |
| `get(id)` | `TimeEntry` | Get by ID (includes costRate/hourlyRate) |
| `update(id, body)` | `TimeEntry` | Update a time entry |
| `delete(id)` | `void` | Delete a time entry |
| `getForUser(userId, params?)` | `TimeEntry[]` | List entries for a user |
| `createForUser(userId, body)` | `TimeEntry` | Create for another user |
| `stopTimer(userId, body)` | `TimeEntry` | Stop a running timer |
| `deleteForUser(userId, ids)` | `TimeEntry[]` | Delete specific entries by ID |
| `bulkEdit(userId, body)` | `TimeEntry[]` | Bulk edit (accepts single or array) |
| `duplicate(userId, id)` | `TimeEntry` | Duplicate an entry |
| `updateInvoicedStatus(body)` | `void` | Mark entries as invoiced |
| `getInProgress()` | `TimeEntry` | Get running timer |

**CreateTimeEntryRequest:**
```typescript
{
  start?: string;          // ISO 8601 — "2026-03-23T09:00:00Z"
  end?: string;            // ISO 8601
  projectId?: string;
  taskId?: string;
  description?: string;
  billable?: boolean;
  tagIds?: string[];
  type?: "REGULAR" | "BREAK";
}
```

**TimeEntry fields:**
```typescript
{
  id, billable, description, projectId, taskId, userId, workspaceId,
  tagIds, kioskId, isLocked, timeInterval, customFieldValues, type,
  costRate?,    // only on GET responses
  hourlyRate?,  // only on GET responses
}
```
